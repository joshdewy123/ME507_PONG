Detailed Description

Main header for ping pong launcher project.

This file contains global includes, definitions, and peripheral handles used throughout the autonomous ping pong ball launcher project.

Core modules include:

  • FSM for control logic
  • IMU (BNO055) for orientation (demo only)
  • Motor control for turret and flywheel
  • Servo control for ball loading

The IMU is functional and reports heading data, but is not used in closed-loop control. This was left as future work.
