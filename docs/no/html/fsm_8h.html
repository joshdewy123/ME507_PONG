Detailed Description

Finite State Machine (FSM) header for autonomous launcher behavior.

This module defines the interface for an FSM that coordinates different states of the ping pong launcher system. These may include manual control, auto-targeting, loading, launching, and safety handling.

The FSM is run in a loop from main.c, and each state should define its own entry actions and transitions.
