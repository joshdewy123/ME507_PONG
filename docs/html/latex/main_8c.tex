\doxysubsection{Detailed Description}
Application entry point for ping pong launcher project. 

This file initializes the STM32 peripherals, sets up the finite state machine (FSM), and starts the main control loop. The IMU is initialized and verified, but not actively used in the FSM during runtime.

Major functions\+:
\begin{DoxyItemize}
\item Setup clocks, timers, I2C, and UART
\item Initialize motors and servo
\item Run FSM in infinite loop
\item Provide IMU debug output for calibration and heading
\end{DoxyItemize}
