Detailed Description

Motor control driver for bidirectional PWM-based DC motors.

This module defines the motor_t struct and function prototypes for controlling brushed DC motors (e.g., turret and flywheel motors) via dual PWM channels.

Used for:

  • Turret aiming motors (Pololu 50:1 DC motors with encoders)
  • Flywheel launcher motors (Maxon DC motors)
